## 👨‍💻 Sobre mí

Soy **Franco Kaddour**, desarrollador frontend de Buenos Aires, Argentina 🇦🇷.  
Me apasiona crear interfaces modernas, funcionales y con un diseño que enamore.  
Mi objetivo es unir **estética + rendimiento** para ofrecer experiencias digitales memorables.

- 🔭 Actualmente trabajando en proyectos personales con **React** y **Sass**.  
- 🎯 Próximo objetivo: dominar **UX/UI Design**.  
- 📚 Siempre aprendiendo nuevas tecnologías y metodologías.  
- ⚡ Fun fact: puedo pasar horas ajustando una animación hasta que se vea perfecta.

## 🧠 Filosofía de trabajo

> “El código es como el humor: cuando tienes que explicarlo, no es bueno.”

- **Calidad sobre cantidad**: prefiero menos features pero bien pulidas.  
- **Diseño centrado en el usuario**: cada decisión visual tiene un propósito.  
- **Aprendizaje continuo**: el mundo tech cambia rápido, y yo con él.
